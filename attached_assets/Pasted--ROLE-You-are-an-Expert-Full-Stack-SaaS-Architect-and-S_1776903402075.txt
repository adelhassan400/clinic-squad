🧠 ROLE

You are an Expert Full-Stack SaaS Architect and Senior React/Firebase Developer.

Your task is to design and generate a complete production-ready SaaS application called:

ClinicSquad

A multi-tenant medical clinic management system with a strict subscription-based access control system.

🏥 PRODUCT OVERVIEW

ClinicSquad is a SaaS platform for clinics that allows doctors to:

Manage patients
Schedule appointments
Track finances
Organize clinic operations

The system is based on a 15-day free trial followed by paid subscription selection between two plans.

💳 SUBSCRIPTION SYSTEM (CORE LOGIC)
⏳ Trial Phase
Every new clinic gets a 15-day free trial
Trial starts immediately after admin activation
No payment required initially
Countdown timer visible in dashboard
🚫 ACCESS CONTROL RULES

The system MUST block access if:

Trial expired AND no subscription selected
Account is paused or deleted by SuperAdmin
Subscription is inactive
Behavior:
Block dashboard access
Show “Subscription Expired” screen
Provide upgrade options
💰 SUBSCRIPTION PLANS (AFTER 15 DAYS)

After trial ends, user MUST choose one of the following plans:

🥉 Plan 1: Basic Plan
Limited features
Patient management only
Appointment system basic
Lower price
🥇 Plan 2: Premium Plan
Full features access
Financial dashboard
Advanced analytics
AI-ready modules (future integration)
Priority support
🎯 Upgrade Flow:
After trial ends → user is redirected to subscription selection page
Must choose a plan to regain access
Payment initially manual (Vodafone Cash)
Admin confirms and activates subscription
🔐 AUTH & ROLE SYSTEM
Roles:
admin (doctor/clinic owner)
secretary
superadmin
Rules:
Only active subscriptions can access dashboard
Role-based access control required
Expired accounts are locked
